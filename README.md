# pantone-table

## Usage

```js
import * as colors from 'pantone-table';

console.log(colors.pantone_100_c); // '#f6eb61'
```

```js
const colors = require('pantone-table');

console.log(colors.pantone_100_c); // '#f6eb61'
```

You can also import only some colors:

```js
import { pantone_100_c, pantone_7407_c } from 'pantone-table';
```
